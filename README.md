# Інвентаризація техніки

Локальна програма обліку техніки та інших предметів компанії.

## Як відкрити

1. Відкрийте файл `index.html` у браузері (подвійний клік).
2. Або у PowerShell:

```powershell
Start-Process "C:\Users\Tecsa-user\Documents\InventoryApp\index.html"
```

## Що вміє

### Склад
- Окремий розділ для предметів, які **ніким не використовуються**
- У картці предмета: «Де знаходиться» → **Склад** або ПБІ співробітника
- Кнопка **На склад** швидко знімає закріплення

### Співробітники
- Кнопка **Предмети** показує всі закріплені за людиною предмети

## Де зберігаються дані

У локальному сховищі браузера (`localStorage`). Дані залишаються на цьому комп’ютері.
Рекомендовано періодично натискати **Експорт JSON**.
